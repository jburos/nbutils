nbutils
===============================

author: Jacki Buros Novik

Overview
--------

Utilities for working with Jupyter notebooks (ipynbs)

Installation / Usage
--------------------

To install use pip:

    $ pip install nbutils


Or clone the repo:

    $ git clone https://github.com/jburos/nbutils.git
    $ python setup.py install
